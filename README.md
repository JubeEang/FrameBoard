# Frameboard Studio

A personal storyboard web app for creative direction work. It runs entirely in the browser, so it can be deployed as a static site on GitHub Pages.

## Features

- Upload reference images for each shot
- Track shot type, angle, movement, duration, location, time, action, and notes
- Reorder, duplicate, edit, and delete shots
- Grid and list views
- Local browser autosave
- Import and export project JSON files
- Print-ready storyboard layout
- Brainstorm notes editor with headers, bold, italic, underline, highlight, and images
- Separate exports for the full project, storyboard board, and brainstorm notes

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Add `index.html`, `styles.css`, `app.js`, and this `README.md`.
3. In the repository settings, open **Pages**.
4. Set the source to your main branch and root folder.
5. Save, then open the GitHub Pages URL when GitHub finishes publishing.

No build step is required.
